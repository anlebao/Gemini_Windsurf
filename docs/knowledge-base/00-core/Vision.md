# ShopERP Vision

> **Transforming Small Business Operations in Vietnam**

## Mission

ShopERP empowers coffee shops, restaurants, and small retailers in Vietnam with an intelligent, offline-first, VAT-compliant business management platform.

## Core Values

1. **Offline First**: Business never stops, even without internet
2. **VAT Compliance**: Ready for Vietnam 2026 tax regulations
3. **Simplicity**: Easy to use, minimal training required
4. **Affordability**: Accessible for small businesses
5. **Intelligence**: AI-assisted operations, not AI-dependent

## Target Users

### Primary
- Coffee shop owners (1-5 outlets)
- Small restaurant operators
- Household businesses (HKD)
- Small retailers

### Secondary
- Chain store managers
- Bookkeepers/accountants
- Tax consultants

## Success Metrics

| Metric | Target |
|--------|--------|
| Daily Active Users | 1,000+ by end of Year 1 |
| Order Processing | 10,000+ orders/day |
| Uptime | 99.9% (offline capability ensures 100% local) |
| User Satisfaction | NPS > 50 |
| Tax Compliance | 100% VAT 2026 ready |

## Strategic Pillars

### 1. Operational Excellence
- Fast order processing (< 2 seconds)
- Kitchen display system integration
- Inventory auto-deduction
- Real-time sync when online

### 2. Financial Compliance
- VAT 2026 ready for both Company and HKD
- Immutable accounting entries
- Period closing with audit trail
- E-invoice integration (future)

### 3. Multi-Channel Presence
- In-store (station apps)
- Mobile apps (customer ordering)
- Web dashboard (analytics)
- Social media integration (future)

### 4. Intelligence Layer
- Sales forecasting
- Inventory optimization
- Customer behavior analytics
- Automated reporting

## Differentiators

| Feature | ShopERP | Competitors |
|---------|---------|-------------|
| Offline Operation | Native | Limited/None |
| VAT 2026 HKD Support | Yes | Rare |
| AI-Assisted Dev | Built-in | No |
| Multi-tenant SaaS | Yes | Mostly on-premise |
| Real-time Sync | NATS-based | Polling/Simple |
