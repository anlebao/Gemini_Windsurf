## B3. Rendering Mode Decision
**Status:** ✓ Documented

**Current State:**
- `Components/Pages/Counter.razor` has `@rendermode InteractiveServer` (line 2)
- `Components/_Imports.razor` imports `RenderMode` (line 6)

**Decision:** **InteractiveServer**

**Rationale:**
- App already uses InteractiveServer in Counter.razor
- Hybrid architecture (Razor Pages + Blazor) works best with InteractiveServer
- No need for InteractiveAuto complexity at this stage
- Static SSR not suitable for accounting UI with real-time updates

**Implementation:**
```csharp
builder.Services.AddRazorComponents()
    .AddInteractiveServerComponents();

app.MapRazorComponents<App>()
    .AddInteractiveServerRenderMode();
```
